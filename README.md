# AI-Based Personalized Diet & Fitness Recommendation System

An AI-powered web application that provides **personalized diet and workout recommendations** using **Google Gemini Generative AI**. The system tailors suggestions based on user preferences, fitness goals, lifestyle, dietary restrictions, and health conditions, offering an interactive and intelligent health assistant.

---

##  Project Overview

Maintaining a healthy lifestyle can be challenging due to busy schedules and lack of personalized guidance. This project addresses the issue by using **Generative Artificial Intelligence** to dynamically generate diet and fitness recommendations tailored to individual users. Unlike traditional systems, this application also includes an **AI chatbot** for real-time health and nutrition queries.

---

##  Features

- Personalized diet recommendations  
- Customized workout plans  
- Meal suggestions (breakfast & dinner)  
- Interactive AI chatbot for follow-up questions  
- Real-time AI-generated responses  
- User-friendly web interface  

---

##  Technologies Used

- **Python**
- **Flask Framework**
- **Google Gemini Generative AI**
- **HTML & CSS**
